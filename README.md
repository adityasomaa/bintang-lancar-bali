# Bintang Lancar Bali

Website resmi **Bintang Lancar Bali** — jasa sedot WC, atasi pipa & saluran mampet, serta kuras septic tank profesional di seluruh Bali. Berdiri sejak 2015, siap 24 jam, bergaransi.

🔗 Referensi domain: [bintanglancarbali.com](https://bintanglancarbali.com)

## ✨ Fitur

- **Preloader** animasi brand saat halaman dimuat
- **Responsif penuh** — mobile, tablet, desktop (hamburger menu)
- **Lenis smooth scroll** (desktop saja; tablet & mobile pakai scroll native)
- **Animasi** halus dengan Framer Motion (reveal on scroll, micro-interactions)
- **Form kontak** yang langsung diarahkan ke **WhatsApp**
- **Tombol WhatsApp mengambang** & peta lokasi
- Placeholder gambar berupa **grafik SVG** (mudah diganti foto asli)
- SEO friendly (metadata, Open Graph, bahasa Indonesia)

## 🧱 Tech Stack

- [Next.js 16](https://nextjs.org/) (App Router)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Lenis](https://lenis.studio/) (smooth scroll)

## 📄 Halaman

| Route       | Deskripsi                                  |
| ----------- | ------------------------------------------ |
| `/`         | Beranda — hero, layanan, keunggulan, dll.  |
| `/tentang`  | Profil perusahaan, visi & misi, nilai      |
| `/layanan`  | Daftar lengkap & detail layanan            |
| `/harga`    | Estimasi harga & paket layanan             |
| `/galeri`   | Galeri dokumentasi pengerjaan              |
| `/kontak`   | Info kontak, peta, & form ke WhatsApp      |

## 🚀 Menjalankan Lokal

```bash
npm install
npm run dev      # http://localhost:3000
```

Build produksi:

```bash
npm run build
npm run start
```
